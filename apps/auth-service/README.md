# auth-service

NestJS authentication service for HEDGE.

## Stack

- NestJS
- Prisma ORM
- PostgreSQL
- Redis
- JWT + Passport
- Argon2
- Docker

## Structure

- `src/auth`
- `src/users`
- `src/sessions`
- `src/audit`
- `src/common`
- `src/config`
- `src/database`
- `src/prisma`

## Commands

- `npm run start:dev`
- `npm run build`
- `npm run prisma:generate`
