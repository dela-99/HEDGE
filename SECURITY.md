# Security Policy

HEDGE handles fintech workloads, so protect customer and operational data by default.

## Secret handling

- Never commit secrets to the repository.
- Store local secrets in untracked environment files or an approved secret manager.
- Rotate any credential immediately if it is exposed in code, logs, or screenshots.

## Reporting

If you discover a security issue, report it privately to the maintainers before opening a public issue or pull request.
